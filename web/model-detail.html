What each flag does:
• --runtime nvidia --gpus all: Use NVIDIA GPU
• --ctx-size 4096: Context window (tokens)
• --n-gpu-layers 99: Offload all layers to GPU
• --flash-attn on: Reduce memory usage 30-50%
• --mlock --no-mmap: Lock model in RAM
• --threads 4: CPU threads for 6-core ARM
