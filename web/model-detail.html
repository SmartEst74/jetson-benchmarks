How to Reproduce These Results

Run these commands on a Jetson Orin Nano Super 8GB with JetPack 6.2:

# 1. Start the model server
docker run -d --name bench-model ...

# 2. Run speed benchmark
python3 scripts/quick-speed-test.py

# 3. Run role benchmarks (all 8 roles × 3 prompts)
python3 scripts/bench-roles-live.py

# 4. View results
cat /tmp/bench-results/all_results.json

How Scoring Works

Each role has 3 prompts with different scoring methods:

🔬 Prompt 1 — Scientific
Automated checklist. Binary pass/fail on specific requirements. 100% reproducible.
Score = (Passed Checks / Total Checks) × 100
🔬 Prompt 2 — Scientific
Automated checklist. Tests specific technical knowledge. 100% reproducible.
Score = (Passed Checks / Total Checks) × 100
🎨 Prompt 3 — Creative
Expert review + community rating. Tests complex problem-solving.
Score = Expert (70%) + Community (30%)
⚠️ Subjectivity Warning
Creative prompt scores (Prompt 3) are subjective. They reflect reviewer opinion. Community ratings help calibrate, but some subjectivity remains. See scoring methodology for details.
