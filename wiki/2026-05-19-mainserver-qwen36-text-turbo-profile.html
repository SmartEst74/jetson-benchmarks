-dev CUDA0,CUDA1
-sm row -ts 5,2 -mg 0

--n-cpu-moe 38

Important trade-off

This profile is text only. It drops -mm mmproj..., so it is not the right default if vision support is required. The safe multimodal baseline remains useful, but it is slower.
